# Ticket Application System

## Project Description
This is a C# Windows Forms Application for calculating bus ticket prices.

## Features
- Enter passenger details (name, age, gender)
- Select travel category (One, Two, or Three)
- Enter travel distance
- Calculate ticket price with automatic discounts
- Clear form functionality
- Professional user interface

## Pricing Rules
| Category | Price per km |
|----------|-------------|
| Category One | R20 |
| Category Two | R35 |
| Category Three | R50 |

## Discount Rules
- Children under 12 years: FREE ticket
- Female passengers: 50% discount

## How to Run
1. Open the project in Visual Studio
2. Build the solution (Ctrl + Shift + B)
3. Press F5 to run the application

## Controls
- TextBox for passenger name
- RadioButtons for gender (Male/Female)
- TextBox for age
- ComboBox for travel category
- TextBox for travel distance
- Buttons: Calculate Ticket, Clear Form, Exit Application

## Project Structure
- Form1.cs - Main form logic
- Form1.Designer.cs - Form design and controls
- TicketApp.csproj - Project file
